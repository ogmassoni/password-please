Quais são as caracteristicas de uma senha segura?

Longa

Quanto mais longa a senha, mais segura ela é. Senhas devem ter pelo menos 10 caracteres.

Complexa

Uma senha forte é composta por uma combinação de letras, números e símbolos para maior segurança.

Exclusiva

Use uma senha única para cada conta, reduzindo a vulnerabilidade em caso de vazamento de dados.
